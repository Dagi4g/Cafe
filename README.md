# Campus Cafe

QR code ordering system for the campus cafe.

---

## What It Does

Students scan QR codes at tables to:
- View menu with real-time availability
- Order food
- Pay digitally or with cash
- Get food delivered to their seat

---

## Built With

Django

---

## Getting Started

```bash
git clone https://github.com/yourusername/cafe.git
cd cafe
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
